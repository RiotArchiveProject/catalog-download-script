# What does this toolset do?

	The script attempts to streamline the process of downloading older versions of Riot Games projects
	It utilizes Python, and rman-dl (https://github.com/moonshadow565/rman)

# Installation

	Once unpacked, place these files and tools into any directory you want, and then install the following software

		Required software

			Python 3.10. I am using 3.10.6 personally
				Download it from here: 
					https://www.python.org/downloads/release/python-3106/
				Install py launcher
				Associate files with Python
				Add Python to environment variables
				Optionally install for all users
				Post-install open command prompt (cmd) and issue the following command
				
					pip install requests
				
			MSVC++ Redistributable packages. These are needed for rman
				
				Download the needed versions from here:
					https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170

# Downloading Setup

	If you have followed the installation instructions, doubleclick download-manager.py and follow the on-screen prompts.

	Updating the catalog can be done by downloading a new one from the maintainer's git page, or data updates on the discord. Simply overwrite the old one.
